🔒 Privacidad y protección de datos

Los datos que introduces en este formulario (nombre, teléfono y dirección) se utilizan exclusivamente para gestionar y entregar tu pedido.

No se almacenan en ninguna base de datos permanente. Permanecen en el sistema el tiempo mínimo necesario para procesar el pedido y garantizar su trazabilidad, con un máximo de 1 hora, tras lo cual se eliminan automáticamente.

Tus datos no serán cedidos a terceros ni utilizados para comunicaciones comerciales, publicidad o spam de ningún tipo.
